React Bootstrap Date&Time Range Picker

描述

基于Bootstrap 3 与 React.js 制作的时间范围选择组件

演示

用法

<DateTimeRangePicker
    style={{width: 400}}
    beginTime={this.state.begin}
    endTime={this.state.end}
    onChange={this.handleChange}
/>

组件属性

名称 类型 默认值 说明
elementType String input 设置触发器的类型，可以为 input 或 button
beginTime Date now 设置选择器的起始时间
endTime Date or String now + 72h 设置选择器的结束时间。如果想要设置为无结束时间，则填Infinite
onChange Function - 选定时间后的回调函数，返回两个参数beginTime 和 endTime。如果无结束时间，第二个参数则返回字符串 Infinite
className String form-control 设置触发器的类
